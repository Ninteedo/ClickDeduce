override val needsBrackets: Boolean

Whether this term should be surrounded by brackets when converted to text inside another term.

Whether this term should be surrounded by brackets when converted to text inside another term.

Default is true, should only be overridden for terms that should not be bracketed in any context. This is common when a term is displayed as a single element, for example a number or variable.

A term is bracketed if this is true and it is being called by toTextBracketed.
