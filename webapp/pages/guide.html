ClickDeduce Guide

ClickDeduce is a tool for learning about and creating evaluation and type-checking rules for simple programming languages.

Kinds of Terms

There are three kinds of terms in ClickDeduce (examples are from the LArith language):

  • Expressions: Basic building block, which can be evaluated to a value or type-checked to a type. Examples: Num, Plus, Times.
  • Values: The result of evaluating an expression that cannot be evaluated further. Examples: NumV.
  • Types: The result of type-checking an expression, the kind of value that an expression would evaluate to. Examples: IntType.
